Ika2.vim
===

Usage
---

The following commands available.

Check stages now and the next box.

* `Ika2StagesNow`
* `Ika2StagesNext`

Check when the specified rule will open.

* For Ranked Battle
  * `Ika2NextZoneG`
  * `Ika2NextRainG`
  * `Ika2NextClamG`
  * `Ika2NextTowerG`
* For League Battle
  * `Ika2NextZoneL`
  * `Ika2NextRainL`
  * `Ika2NextClamL`
  * `Ika2NextTowerL`
